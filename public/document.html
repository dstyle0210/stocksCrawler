[표로이동]

참고용어설명

- 기준순수익 : 최근 4분기 예상순수익의 합 - 예상배당금

- 기준적정주가 : 기준순수익과 잔여이익을 합하고, 현재 주식수로 나눈것

- 기대수익률 : BBB- 등급 채권 5년 이자률 , 주식 리스크를 BBB- 채권과 동일시 함

- 국고채 3년 : 예금이자율(무위험 현금가치)로 판단

ROE ( 자본대비수익률 )

- 표시방법 : 기준순수익에서 최근 표시 지배자본으로 ROE 산출

- ROE는 영업이익과 똑같이 생각하면 됨(높을수록 좋음)

- 기대수익률(BBB- 등급 채권 5년 이자률)보다 높아야 함.

배당률

- 표시방법 : 전일주가 기준에서 예상 배당금에 대한 배당률 계산

- ROE보다 작아야 한다.

- 국고채 3년 보다 높아야 한다.

- 번것(ROE)보다 크게 준다 라는것은 곳간(잉여현금) 빼서 준다라는것.

- 리스크가 있는데, 무위험 예금이자(국고채 3년)보다는 최소 높아야 한다.

- 가장 이상적인건 ROE의 절반 정도가 좋다. (이익의 반은 나눠준다 라는 뜻)

안전마진

- 표시방법 : 전일주가에서 기준적정주가 까지의 비율

- 최대한의 기대상승폭 으로 생각해도 좋음

- 절대적인것은 없고, 과도한 차이가 나는것도 이상한것

RIM(잔여이익)

- 표시방법 : 기준 ROE와 기대수익률을 기준한 잔여이익 비율

- 음수(-)라는 것은 내년에 기대수익률 만큼 발전하지 못한다는것

- RIM이 양수라 해도, 무엇으로 양수를 만들어 냈는지는 확인이 필요함

외인(6일)

- 표시방법 : 전일까지의 최근 6일간의 외인수급의 합

- 양수로 유지 되고있다면, 계속해서 외인이 매집중이라는 의미

- 음수 -> 양수로 전환 후, 일주일 정도 격차로 보는것을 추천
